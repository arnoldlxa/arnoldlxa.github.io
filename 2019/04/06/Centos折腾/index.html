刻录和安装

1 u盘安装
Ceontos7镜像支持GPT格式的磁盘，将磁盘格式转化为GPT，Centos安装将会格式化以前磁盘的内容，至于怎样不格式化安装@我哈
使用rufus刻录Centos7.DVD到u盘,使用PTG,可以使用超过4个分区的主分区,不用去使用逻辑分区.安装选项中点重 工作
然后选择自己配置磁盘.我的磁盘1T,划分如下:

  • /boot/b 1G
  • / 80G
  • swap 2G
  • /home 400G
  • /backfile 300G
  • /worksplace 153G
    2 安装期间设置root密码和普通用户名和密码.安装完成后重启进入，会有一个用户协议同意书，反正我的电脑是以命令行的格式显示，其中先选择“1”,然后输入“a”(为accept的简写)，在选择“2”，同样，之后是“c”（就是continue).之后电脑重启进入图形界面.

Ceontos7基本配置

设置sudo

终端中以root身份输入命令
# visudo
在配置文件中找到下面的几行内容：
## Allow root to run any commands anywhere
root ALL=(ALL) ALL

然后，在上行内容下面添加下面内容
xxxx ALL=(ALL) ALL
其中的xxxx改成自己的用户名即可，这样以后就可以用sudo命令了

在CentOS7中安装思维脑图软件XMind

解压deb包，得到data.tar.gz 和control.tar.gz 两个归档文件
解压data.tar.gz，得到一个usr文件夹，把此文件夹下的lib/xmind文件夹复制到/opt/文件夹下，share子文件夹移动到/usr/文件夹下和已有的share文件夹合并。命令如下：
$ cp -r usr/lib/xmind/ /opt/
$ cp -r usr/share/ /usr/
解压control.tar.gz，执行解压出来的postinst文件：
$ ./postinst
编辑/usr/share/applications/xmind.desktop文件内容为以下内容（只修改了Exec和Icon）
$ sudo gedit /usr/share/applications/xmind.desktop
Icon=/usr/share/pixmaps/XMind.png
Exec=/opt/xmind/XMind
