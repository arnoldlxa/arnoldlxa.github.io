安装 npm

如果node不是最新的，node有一个模块叫n，是专门用来管理node.js的版本的。使用npm（NPM是随同nodejs一起安装的包管理工具）安装n模块
$ sudo npm install -g n
然后，升级node.js到最新稳定版
$ sudo n stable 或者 sudo n v7.6.0
