1 usb制作u盘启动：

Linux Mint安装磁盘以MBR格式最好，当转为GPT格式时，安装会出现.

the partition table format in use on your disks normally requires you to create a separate partition for boot loader code. this partition should de marked for use as a “reserved bios boot area” and should de at least 1 mb in size. note that this is not same as a partition mounted in /boot.

下载linux mint 镜像包，用rufu刻录进入u盘，选择MBR格式刻录，刻录完毕后，进入BOIS设置USB启动优先于硬盘启动，重启（Ctrl + alt + delete），进入安装界面，选择语言，中文，以其他方式安装，分配原则：
/ 分区，为主分区，50G左右，因为根分区不宜过大，损坏后会导致系统出问题。
交换分区，2G，原则上不超过2G，因为虚拟的硬盘内存的读写速度太差，多了效果不好，还浪费硬盘。
/home，我的经验尽可能的多的分配给/home目录硬盘空间，因为你的工作目录就是在这个下面的，比如/home/linuxtest（你的用户名）下你的桌面文档，下载，图片等
/backfile 专门放备份文件的。
/ 随便一个文件名啦 其他的剩余的硬盘可以放在另一个分区挂载一个文件下。
接下来就是等待安装啦，记住不要联网，否则会安装下载文件很久滴。
